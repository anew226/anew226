Here's Why I am Excited about the Web

First, I should point out that a large part of the discussion about developing a web app has revolved around the Return on Investment (ROI).

However, there are so many more benefits to developing a web app that I'd like to share with you. Here are a few things I get excited about when it comes to web applications.

1. Benefits of Web Applications:

In the article 'What Is a Web Application? How It Works, Benefits and Examples' the Indeed Editorial Team spells out the benefits of web apps.

A web application has many benefits:

  • You don't need to install it on the hard drive, so it doesn't cause space limitations.
  • It requires less support and maintenance from the business and lower technical requirements from the user's computer.
  • A web application reduces costs for both the end-user and the business.
  • Web applications are always up to date because updates are applied centrally.
  • You can access web applications anywhere with a web browser.
  • All users can access the same version, so it eliminates any compatibility issues.
  • As long as the browser is compatible, web applications can run on multiple platforms regardless of the operating system or device.
  • Web applications discharge the developer of the responsibility of building a client that is compatible with a specific type of computer or a particular operating system.
  • Web applications reduce software piracy in subscription-based web applications.

2. Intuitive UI Design

So far, with the Quantum Web App, most of the meetings with the technicians at AJWT have focused on the design of the User Interface. That's because it's advantageous to create an app that works the way you would expect it to. It's fundamental to the way we interact with the app. So, developing a good UI is the first, fundamental place to start when designing an app.

Using an app with an intuitive interface saves you time and frustration. With a well-designed UI, there's no need to spend a lot of time learning or remembering how to use it. Well-designed UIs are relatively simple and largely self-explanatory. Interactions have clear, obvious labels that help a user immediately understand what to do.

You can also design a workflow that prevents obvious 'footguns'

  • For example, preventing a tech from changing the WO status to Costing before writing the Tear Down / Inspection Report

When it comes to our app, having a good UI will be most beneficial to the new technicians. Currently, new techs spend a good deal of time learning the ins and outs of Quantum. This means that there is potential to save a lot of time. Imagine having an app that a new technician could pick up and learn in 20 minutes. I'm sure the cell leads wouldn't complain if there were fewer mistakes to fix because the UI prevented new techs from making obvious errors.

3. The power of APIs

What is an API?

APIs (Application Programming Interfaces) are a great way to add functionality to your web app.

An API is a set of defined rules that explain how computers or applications communicate with one another.

Application Programming Interface (API) IBM Cloud

For example, if you wanted to build a weather app, you can take advantage of one of AccuWeather's APIs.

AccuWeather's APIs have:

  • The largest and best collection of real-time weather data
  • Expert data analysis
  • Forecasts with proven Superior Accuracy

Instead of going out on your own to set up weather sensors and designing your own data analytic tools, you can easily add functionality to your weather app by connecting to one of AccuWeather's APIs.

Advantages of using APIs:

  • Innovation => APIs enable businesses to respond quickly to expand their services by easily adding functionality
  • Integration => APIs enable seamless integration between apps
  • Data => APIs allow data to be shared easily across different platforms via website, Android, iOS, etc
  • Security => APIs create an extra layer of protection between your data and server

So, it shouldn't be too surprising to find out that our Quantum Web App will connect to Quantum using the API Module for Quantum Control. This will enable us to connect our app to Quantum.
